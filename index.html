Evrimo – An Intelligent vehicle routing software

The beta version of Evrimo is equipped with an intelligent vehicle routing algorithm that finds optimal vehicle routing paths in different cities of the world. The user provides the geocodes of several locations in a single path. It then computes the optimum vehicle routing sequence for all the locations in a path. Finally the optimum visit sequence is generated as output in different standard display formats. Evrimo can be used for variety of application such as optimization of distribution paths for goods and consumables, home delivery services, postal delivery services, transportation and many other related services.

Features

The beta version of Evrimo is equipped with the following features.

Multiple output display options

The optimized vehicle route can be viewed on different devices and softwares. It can be viewed on Google Earth through a KML file, on Sygic through ITF file, or as simple text file. The KML file shows the detailed street paths on Google Earth desktop application. The ITF file can be downloaded on a mobile phone with Sygic for detailed GPS based navigation. The text file lists down the sequence of stops in their visit sequence along with total distance in Kilometers. All these files are automatically generated and can also be downloaded on your device for future reference.

Multiple Input Options

The inputs to Evrimo include path details such as city name, path names and geocodes of locations and warehouse locations in a particular path, and other miscellaneous details. The user can give inputs to Evrimo through form based interface or can upload bulk data through CSV files. The bulk upload option can be used to easily import existing data into Evrimo database.

Maintain History

Maintains database for each user. All the inputs, output files, profile information and billing related details are maintained in secure database servers.

Paths with numerous destinations supported for multiple cities

Each user can optimize vehicle paths in any city of the world that are supported by Evrimo. For a standard user, up to 50 destinations can be included in a single path. On customized request, the number of destinations can be increased up to 100 per path. Moreover, the number of destinations can also be increased beyond 100, but at the expense of increased computation delay.

Customization and support

Evrimo team also provides customized solutions and support on request. The customization and support services may include adding new cities to Evrimo, increasing number of destinations in a particular path, helping you to optimize distribution paths for your business, integration of Evrimo service with your online solution and many more.
